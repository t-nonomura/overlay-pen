# プライバシーポリシー

最終更新日: 2026-04-01

本プライバシーポリシーは、Android アプリ「最前面ペン | 画面に描ける注釈ツール」（英語名: Overlay Pen / Screen Pen、以下「本アプリ」）に適用されます。

## 1. 基本方針

本アプリは、画面上に注釈や描画を重ねて表示する機能を提供します。  
本アプリは、ユーザーの個人情報をできる限り収集しない設計を基本方針としています。

## 2. 取得する情報

現時点の本アプリは、以下の情報を開発者のサーバーへ送信、収集、共有しません。

- 氏名、住所、電話番号、メールアドレスなどの個人情報
- アカウント情報
- 位置情報
- 連絡先
- 写真、動画、音声などのメディアファイル
- 広告 ID
- 解析用の利用ログ
- 購入情報

## 3. アプリ内で扱う情報

本アプリは、ユーザーが画面上に描いた線、色、太さ、不透明度などの描画状態を、アプリの動作中に一時的に端末メモリ上で扱います。

現時点では、これらの情報について以下の仕様です。

- 端末外への送信は行いません
- 開発者または第三者と共有しません
- セッション終了時に消去されます
- 永続保存機能は提供していません

## 4. 利用する権限

### オーバーレイ表示権限

他のアプリの上に描画 UI を表示するために、Android のオーバーレイ関連権限を利用します。

### フォアグラウンドサービス

描画用オーバーレイを安定して表示するために、Android のフォアグラウンドサービスを使用します。

## 5. 第三者提供

本アプリは、ユーザー情報を第三者へ販売、共有、提供しません。  
現時点では広告 SDK、アクセス解析 SDK、外部課金 SDK、クラッシュレポート SDK も組み込んでいません。
