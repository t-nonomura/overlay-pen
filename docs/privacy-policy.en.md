# Privacy Policy

Last updated: 2026-04-01

This Privacy Policy applies to the Android app "最前面ペン | 画面に描ける注釈ツール" (working English name: Overlay Pen / Screen Pen, the "App").

## 1. Overview

The App provides a floating drawing and annotation tool that lets users draw over the screen.  
The App is designed to minimize the collection of user data wherever possible.

## 2. Data We Collect

At this time, the App does not send, collect, or share the following data with the developer's servers:

- Personal information such as name, address, phone number, or email address
- Account information
- Location data
- Contacts
- Photos, videos, audio, or other media files
- Advertising ID
- Analytics data
- Purchase information

## 3. Data Processed on Device

The App temporarily handles drawing-related state in device memory while the App is in use, such as:

- Strokes drawn by the user
- Selected color
- Stroke width
- Opacity
- Tool state

At this time:

- This data is not transmitted off the device
- This data is not shared with the developer or third parties
- This data is cleared when the drawing session ends
- The App does not currently provide persistent save functionality

## 4. Permissions and Device Features

The App uses the following Android permissions or capabilities.

### Overlay Permission

The App uses Android overlay-related permission so it can display drawing controls over other apps.  
This is required for the App's core function: drawing on top of the screen.

### Foreground Service

The App uses an Android foreground service to keep the overlay drawing feature active and stable while in use.

## 5. Data Sharing

The App does not sell, share, or provide user data to third parties.

At this time, the App also does not include:

- Advertising SDKs
- Analytics SDKs
- Billing SDKs
- Crash reporting SDKs

## 6. Children's Privacy

The App is not primarily directed to children under the age of 13.

## 7. Security

The App is designed to reduce privacy risk by avoiding transmission of user data off the device.  
If future versions add save, export, sharing, billing, analytics, or similar features, this Privacy Policy will be updated as needed.
